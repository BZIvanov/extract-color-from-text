Auth

Register

  - register new user

POST /api/v1/auth/register

{
  "name": "Ina",
  "email": "ina@mail.com",
  "password": "12345678",
  "role": "user"
}

Login

  - login already registered user

POST /api/v1/auth/login

{
  "email": "ina@mail.com",
  "password": "12345678"
}

Logout

  - logout user

GET /api/v1/auth/logout
